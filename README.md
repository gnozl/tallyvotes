# tallyvotes

How to run:
 1. Install python 3

Mac
``python3 rankedvoting.py``

Windows
``py rankedvoting.py``

votes.csv is structured as

``"VoterID","1st Place","2nd Place","3rd Place"``

dq.txt contains the names of any disqualified ballots or poems. The names must match those in the votes.csv file exactly.
